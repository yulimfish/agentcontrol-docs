# AgentControl Docs Agent 指南

这个仓库包含 AgentControl 项目族的文档。

## 职责

- 解释 `AgentControl-Fabric` 和 `AgentControl-MCP` 如何配合工作。
- 明确安全边界。
- 保持安装说明与当前端点名称、工具名称和构建产物一致。
- 记录同时影响 Fabric 和 MCP 的变更。

## 相关仓库

- `AgentControl-Fabric`：本地 Minecraft 客户端模组。
- `AgentControl-MCP`：面向 AI 的 MCP 服务。
- `AgentControl`：包含所有项目的总仓。

## 文档规则

- 不要暗示 AgentControl 能绕过服务器规则。
- 示例优先使用当前 Fabric 驱动的控制路径。
- 说明 Mod Menu 是可选依赖，而不是必须依赖。
- 安装说明应按用户角色拆分：最终用户、MCP 使用者、开发者。
